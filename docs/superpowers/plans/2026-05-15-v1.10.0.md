# v1.10.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a minor release that fixes a critical LanceDB migration blocker, corrects list-sort accuracy across all entity ops, fixes driver PWA scroll/pagination/layout bugs, updates stale asset cache-busting, and redesigns dispatch list views for operational clarity.

**Architecture:** All backend changes are isolated to DB ops files (remove the `limit` cap from fetch queries so in-memory sort has the full row set). All frontend changes are vanilla JS/CSS — no build step, no framework. Issues #155 and #154 are already committed to main; this plan closes them without re-implementing them.

**Security note:** All frontend tasks that build dynamic HTML use the existing `escHtml()` utility from `static/dispatch/app.js` for every field that originates from API responses. Never insert raw API string data into rendered HTML without `escHtml()`.

**Tech Stack:** Rust (Axum, LanceDB 0.27), vanilla JS ES modules, CSS custom properties

- [ ] **Step 4: Add frontend pagination to `loadTrips`**

In `static/driver/pages/trips.js`, replace the entire `loadTrips` function (lines 79–124) with the version below. The key changes: capture `allItems` from the response, define `PAGE_SIZE = 10`, call `showMore()` on first render and again on each button click, remove and re-add the "Load more" button as needed.

```js
async function loadTrips(tab) {
  currentTab = tab;
  tripList.innerHTML = '';

  const loadingEl = document.createElement('div');
  loadingEl.className = 'trips-loading';
  const spinner = document.createElement('div');
  spinner.className = 'spinner';
  loadingEl.appendChild(spinner);
  tripList.appendChild(loadingEl);

  try {
    const data = await apiFetch(`/trips?tab=${tab}`);
    if (currentTab !== tab) return;

    tripList.innerHTML = '';

    const allItems = (data && data.items) ? data.items : [];
    if (allItems.length === 0) {
      const emptyEl = document.createElement('div');
      emptyEl.className = 'trips-empty';
      emptyEl.textContent = tab === 'current' ? 'No current trips' : `No ${tab} trips`;
      tripList.appendChild(emptyEl);
      return;
    }

    const PAGE_SIZE = 10;
    let shown = 0;

    function showMore() {
      const slice = allItems.slice(shown, shown + PAGE_SIZE);
      slice.forEach(trip => {
        const card = renderTripCard(trip, tab);
        tripList.appendChild(card);
      });
      shown += slice.length;

      const existingBtn = tripList.querySelector('.load-more-btn');
      if (existingBtn) existingBtn.remove();

      if (shown < allItems.length) {
        const btn = document.createElement('button');
        btn.className = 'btn btn--secondary load-more-btn';
        btn.style.cssText = 'display:block;margin:1rem auto;';
        btn.textContent = `Load more (${allItems.length - shown} remaining)`;
        btn.addEventListener('click', showMore);
        tripList.appendChild(btn);
      }
    }

    showMore();
  } catch (err) {
    if (err.status === 401) {
      clearAuth();
      window.location.replace('/driver');
      return;
    }
    tripList.innerHTML = '';
    const errorEl = document.createElement('div');
    errorEl.className = 'trips-error';
    errorEl.textContent = err.message || 'Failed to load trips';
    tripList.appendChild(errorEl);
  }
}
```
